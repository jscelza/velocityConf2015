Demo Prerequisites

In order to run the demos on this USB, you will need Vagrant and Virtualbox. For your convenience the latest installations for the most common platforms are included on this USB.

Vagrant

The demos on this USB has been tested to work with Vagrant version 1.3+. Vagrant version 1.7.2 is provided on this USB for your convenience. Please check out Vagrant Downloads for the latest and/or for instructions.

Platform File
Mac OSx vagrant_1.7.2.dmg
Windows Universal vagrant_1.7.2.msi
Debian 32 bit vagrant_1.7.2_i686.deb
Debian 64 bit vagrant_1.7.2_x86_64.deb
Linux RPM 32 bit vagrant_1.7.2_i686.rpm
Linux RPM 64 bit vagrant_1.7.2_x86_64.rpm

VirtualBox

The virtual machine provided on this USB has been tested to work with Oracle's open source VirtualBox version 4.1.18+. Version 4.3.x is provided on this USB for your convenience. Please check out VirtualBox Downloads for the latest VirtualBox or for instructions.
